Active Security Labs

Windows Security

Windows 11 Local Privilege Escalation(LPE)

This assessment examined a Windows Local Privilege Escalation scenario involving insecure service directory permissions aligned with CWE-276. Through controlled testing in an authorized lab environment, a Binary Hijacking technique demonstrated how improper Access Control List (ACL) configurations can allow privilege boundary violations and lead to SYSTEM-level code execution.

Business Value: This lab highlights the importance of Principle of Least Privilege (PoLP), Access Control auditing, File Integrity Monitoring (FIM), and secure service path management in enterprise environments.

OWASP Juice Shop API Security Assessment

Critical Endpoint Assessment

A structured application security assessment of OWASP Juice Shop identified multiple high-risk vulnerabilities affecting authentication, authorization, and object access controls. Testing included validation of SQL Injection, Broken Object Level Authorization (BOLA), and Insecure Direct Object Reference (IDOR) scenarios, demonstrating how application logic flaws can expose sensitive customer information.

Business Value: The assessment reinforces the importance of robust server-side authorization controls, secure identity management, and the adoption of indirect object references such as UUIDs to protect user data.
